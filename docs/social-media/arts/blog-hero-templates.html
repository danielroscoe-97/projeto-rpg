Como Exportar

1. Abra este arquivo no navegador
2. Selecione cada .hero individualmente
3. Use DevTools → captura de area (Ctrl+Shift+P → "Capture area screenshot")
4. Ou use extensao GoFullPage e recorte cada hero em 1280x720
5. Salve em public/art/blog/heroes/ com o nome indicado no label
6. IMPORTANTE: Mostre ao Dani_ para aprovacao antes de publicar!
